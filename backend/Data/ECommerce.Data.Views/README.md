### Database Engineer Creates View:
1. Backend Developer requests a view (e.g., "I need a ProductListView with id, name, price, category")
2. Database Engineer creates SQL view in migration
3. Database Engineer runs LLBLGen Pro Designer
4. LLBLGen generates view DTO classes in this project
